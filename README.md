# Yarn-e-Commerce

Yarn-e-Commerce is an e-commerce project focused on building a modern multilingual online store for yarn and related products.

## What we do

The organization brings together product management, backend, frontend, QA, DevOps, design, and analytics workflows in one structured environment.

## Main directions

- Product management and delivery process
- Backend API development
- Frontend development
- Admin panel development
- QA manual and automation
- DevOps and infrastructure

## Working approach

We organize work through a unified GitHub Project board, team-based repository access, and shared issue templates and process documentation.

- Backend
- Frontend
- Admin panel frontend
- QA repositories
